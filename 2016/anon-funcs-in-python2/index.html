Anonymous Functions in Python 2

January 30, 2016

An anonymous function is a function that isn't bound by any name. Python provides a way to create anonymous functions by using the lambda keyword. It can also be used with other functions such as map() and filter().

A simple lambda function can be something that divides a number by 3.

Here we see that the lambda function is passed to a variable titled anonfunc(k), with a single parameter defined as k. You could have a number of parameters as with a normal python function. Should also be noted that this lambda function isn't evaluated until anonfunc(k) is called.

Let's see more lambda action. Say we want to iterate over a list, multiply each content by 2, and only show values that have an exact multiple of 3.

This is how it is usually done the "for loop" way.

And this uses lambdas with filter() and map().

As you can see, the lambda functions are being passed into the filter() and map() functions. The map() function takes each element of the list of numbers, applies the lambda function k * 2, and returns a list with the result. Next the filter() function applies the True or False lambda function k % 3 == 0 to each element of the list returned by the map() function. A new list is then returned with each element found True.
